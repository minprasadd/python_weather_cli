# 🌤️ Python_Weather_CLI

A command-line weather application built with Python that fetches real-time weather data and a 5-day forecast using the Open-Meteo API.

---

## 🚀 Features

- Search weather by city name  
- Real-time temperature, humidity, wind speed, and precipitation  
- “Feels like” temperature analysis  
- Weather condition descriptions with icons  
- 5-day weather forecast  
- Clean and readable CLI output  
- Error handling for network and API issues  

---

## 🛠️ Technologies Used

- Python  
- Requests library  
- Open-Meteo API  
- JSON handling  
- Command Line Interface (CLI)
